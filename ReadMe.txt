Hi everybody!

Willkommen zu unserer Version von Hangman!

Hier geht es darum Tina the Turle oder Franklin the Frog vor einem grausamen Tod zu retten, indem ihr entweder die Namen 
verschiedener Schildkröten- oder Froscharten erratet. 

Das Spiel ist im Gradle Menü auf der rechten Bildschirmseite unter hangmanTurtles-Tasks-application mit "Run" zu starten.

Zum Projekt selbst: in unserem Source Ordner findet ihr in dem Ordner Java unsere verwendeten Klassen und in Resources JavaFX 
Datein, selbstgezeichnete Bilder und Animationen, die Wordlists,Schriftarten und die verwendete CSS Datei.

Viel Spaß beim Spielen! 
Rettet Tina und Franklin!

- The Turles :)
